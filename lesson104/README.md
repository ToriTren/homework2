# Настройка Prettier для проекта

## Установка расширения

Убедитесь, что у вас установлено расширение Prettier - Code formatter в VSCode.

## Конфигурация

В проекте созданы следующие файлы конфигурации:

- `.prettierrc` - основной файл конфигурации Prettier
- `.prettierignore` - список файлов и папок, которые будут игнорироваться при форматировании
- `.vscode/settings.json` - настройки VSCode для интеграции с Prettier

## Функциональность

- Форматирование при сохранении файла включено
- Prettier установлен как форматтер по умолчанию
- При сохранении файла автоматически применяются правила форматирования

## Использование

Чтобы вручную отформатировать файл, используйте сочетание клавиш `Shift+Alt+F` (или `Shift+Option+F` на Mac) или контекстное меню "Format Document".

Для форматирования всего проекта можно использовать команду:

```bash
npx prettier --write .
```

## Примечание

Если Prettier не работает, проверьте, что в VSCode выбран правильный форматтер по умолчанию. Нажмите `Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac) и введите "Format Document With...", затем выберите Prettier.
